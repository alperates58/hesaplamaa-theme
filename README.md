# Hesaplamaa Theme

Custom WordPress theme for Hesaplamaa.

## GitHub Updates

The theme includes a GitHub update panel under:

`Appearance > Hesaplamaa Tema`

Set:

- Repository: `alperates58/hesaplamaa-theme`
- Branch: `main`
- Token: leave empty for a public repository, or use a GitHub token for a private repository.

After pushing changes to GitHub, open the panel and use `GitHub'dan Guncelle` on the WordPress site.
